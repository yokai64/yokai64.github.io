The Great Minecraft Experiment

Back to top

This server is an experiment to see how far a single world can be taken through the retro and modern versions of Minecraft. We will be starting with the earliest version of Minecraft that supports multiplayer servers - version a1.0.15 - and slowly progress to the most recent version of Minecraft (whatever that happens to be, currently 1.19). How will the world generation fail? Will we see crazy chunk errors? Who knows!
