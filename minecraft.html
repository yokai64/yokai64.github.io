This server is an experiment to see how far a single world can be taken through the retro and modern versions of Minecraft. We will be starting with the earliest version of Minecraft that supports multiplayer servers - version a1.0.15 - and slowly progress to the most recent version of Minecraft (whatever that happens to be, currently 1.19). How will the world generation fail? Will we see crazy chunk errors? Who knows!

The Problem: Classic Minecraft

You may be thinking: "What is the need for this section? All I need to do is just go into the Minecraft Launcher, and add the version of Minecraft the server is currently on!" That was my thinking too, however after some testing I have found that on modern systems every version of Minecraft prior to 1.3.1 has difficulty running. Not performance wise, but in the sense that you can't even get past the slash screen to join a server or start a new world. You can give it a try yourself, and if you look at the logs you'll see that the game gets stuck trying to load some legacy Java code that the current versions of Windows no longer supports (I haven't tried this on macOS or Linux, so if it works for you via the official launcher then good for you ðŸ™‚)

Fig 2: Trying to run Classic Minecraft via MultiMC and the message Minecraft gets stuck on. The official launcher has the same issue.

I tried a bunch of stuff that I could think of: I ran Java in Windows Compatibility Mode; I downloaded an ancient, really sketchy version of Java to see if that had the old code that I needed; I ran that version of Java in Compatibility Mode. Nothing worked at all.

After some research, I found a small community that is currently running an alpha server. I hopped into their server and discoverd a channel that proposed a solution to running classic Minecraft on modern systems.

(Credits: shoutouts to the AlwaysAlpha.net Discord Server)


The Solution: Betacraft Launcher

There exists an unofficial unrelated-to-Mojang launcher that allows you to launch pre-alpha, alpha and beta versions of the game. It closely resembles early versions of the launcher, so you might get a kick of nostalgia out of it.

You can download the launcher by clicking this link. Download the specific version you require (a Windows .exe or a universal JAR file) and open the launcher. If you want to make sure your inventory remains throughout the server upgrades then click on "Log in" to linking your Microsoft account. You Minecraft Username will automatically be filled in and be set every time you start the game.

Fig 3: Home page of the Betacraft Launcher

Please be aware: The inclusion of Betacraft in this tutorial is not an endorsement of it. The launcher allows you to download Minecraft without logging into an account. Please only use the launcher if you have bought an paid for the full version of Minecraft Java. On top of this, I haven't properly vetted the software, so I cannot confirm the cyber security of the software. If you don't feel comfortable signing in, then please don't, but also be aware I cannot guarantee that your progress will carry over to when we can use the official launcher
